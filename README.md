## 🎯 ฟังก์ชันหลักของระบบ

### 1. การปรับขนาดตัวอักษร (Increase / Decrease Font Size)
เว็บไซต์รองรับการปรับเปลี่ยนขนาดตัวอักษรบนหน้าจอ โดยผู้ใช้งานสามารถปรับเพิ่มหรือลดขนาดของข้อความได้อย่างอิสระ ฟังก์ชันนี้ถูกพัฒนาขึ้นเพื่ออำนวยความสะดวกให้แก่ผู้ใช้งานทุกกลุ่ม โดยเฉพาะอย่างยิ่งกลุ่มผู้ปกครอง ผู้สูงอายุ หรือผู้ที่มีความจำเป็นต้องใช้ตัวอักษรขนาดใหญ่ เพื่อให้สามารถอ่านเนื้อหาและเข้าถึงสารสนเทศบนเว็บไซต์ได้อย่างชัดเจนและมีประสิทธิภาพสูงสุด

### 2. การปรับธีมเว็บไซต์ (Light and Dark Theme)
ผู้ใช้งานสามารถเลือกสลับรูปแบบการแสดงผลของเว็บไซต์ระหว่างโหมดสว่าง (Light Theme) และโหมดมืด (Dark Theme) ได้ตามความเหมาะสมของสภาพแวดล้อม ฟังก์ชันนี้นอกจากจะช่วยยกระดับประสบการณ์การใช้งานและช่วยถนอมสายตาเมื่อต้องใช้งานหน้าจอในสภาวะที่มีแสงน้อยแล้ว ยังสะท้อนถึงภาพลักษณ์การออกแบบเว็บไซต์ที่ทันสมัยและได้มาตรฐานสากลอีกด้วย

### 3. รองรับระบบสองภาษา (Translation)
เพื่อเป็นการรองรับความเป็นสากล เว็บไซต์จึงมีฟังก์ชันสลับภาษาที่สามารถแสดงผลได้ทั้ง ภาษาไทย และ ภาษาอังกฤษ อย่างเต็มรูปแบบ เพื่อตอบสนองความต้องการและอำนวยความสะดวกในการสืบค้นข้อมูลให้แก่นักเรียน นักศึกษาต่างชาติ ตลอดจนผู้ปกครองและหน่วยงานภายนอกที่ต้องการเข้าถึงข้อมูลของภาควิชา

### 4. การจัดการข้อมูลของข่าวสารและอาจารย์
ระบบสำหรับผู้ดูแลเว็บไซต์ (Administrator) ในการบริหารจัดการเนื้อหา เช่น การประกาศข่าวสารของภาควิชา และการปรับปรุงข้อมูลทำเนียบคณาจารย์ให้เป็นปัจจุบัน โดยผู้ดูแลระบบที่ได้รับอนุญาตสามารถเข้าสู่ระบบเพื่อจัดการข้อมูลได้ผ่านช่องทางดังต่อไปนี้
