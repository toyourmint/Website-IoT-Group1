## 🎯 ฟังก์ชันหลักของระบบ

### 1. การปรับขนาดตัวอักษร (Increase / Decrease Font Size)
ผู้ใช้งานสามารถกดเพิ่มหรือลดขนาดตัวอักษรได้ทันที ซึ่งเหมาะมากสำหรับผู้ปกครอง หรือคนที่ต้องการอ่านข้อความขนาดใหญ่ขึ้น

### 2. การปรับธีมเว็บไซต์ (Light and Dark Theme)
สามารถสลับระหว่างโหมดสว่างและโหมดมืดได้ ฟังก์ชันนี้ช่วยเพิ่มประสบการณ์ใช้งาน และทำให้เว็บดูทันสมัยมากขึ้น

### 3. รองรับระบบสองภาษา (Translation)
เว็บไซต์สามารถเปลี่ยนภาษาได้ทันทีระหว่างภาษาไทยและภาษาอังกฤษ เพื่อรองรับนักเรียน นักศึกษาต่างชาติ ผู้ปกครอง บุคคลากรภายนอกที่ต้องการข้อมูล

### 4. การจัดการข้อมูลของข่าวสารและอาจารย์
สามารถเข้าสู่ระบบ Backend ได้ผ่านลิงก์ https://67011430.consolutechcloud.com/admin หรือ https://website-iot-group1-production.up.railway.app/admin
